# Set Up

### Clone the repo.

```
git clone https://github.com/${GitHub Username}/learning-series.git
```

<br>

### Open Terminal on the Folder

```
cd learning-series
```

<br>

### Install the Dependencies

```
npm install
```

<br>

### Run the UI

```
npm run dev
or
yarn dev
```
